## **1. Git的基础命令**

. mkdir 文件夹名称 在当前目录下创建文件夹
. touch 文件名称   在当前目录下创建文件
. ls               查看当前目录下的文件
. ls -a            查看当前目录下的隐藏文件
. cd 文件夹名称    进入某个文件夹
### git 提交
1. git add -A 
2. git commit -m "修改内容注释"
3. 第一次提交 git push origin main 第二次提交 git push
